# 📉 Amazon Egypt Price Tracker Bot

بوت تلجرام لتتبع أسعار أمازون مصر

## الميزات
- تتبع أسعار أمازون مصر
- إشعارات فورية عند انخفاض السعر
- نشر العروض في القناة بلينكات affiliate
- خطة مجانية (5 منتجات) ومدفوعة (غير محدودة)
- نظام دفع عبر InstaPay

## نشر على Railway

1. ارفع الكود على GitHub
2. افتح railway.app واعمل New Project من GitHub
3. أضف Volume وربطه بـ `/app/data`
4. في Settings → Variables أضف:
   - `BOT_TOKEN` = توكن البوت
   - `ADMIN_ID` = الـ chat ID بتاعك
5. Deploy!

## الأوامر

- `/start` - بداية البوت
- `/stats` - إحصائيات (للأدمن فقط)
